# Referent

Приложение для анализа англоязычных статей с помощью AI на Next.js

## Описание проекта

Приложение получает URL англоязычной статьи, выполняет парсинг и предоставляет три функции анализа через AI:
- **"О чем статья?"** - краткое описание статьи (2-3 предложения)
- **"Тезисы"** - основные тезисы в виде нумерованного списка
- **"Пост для Telegram"** - форматированный пост с эмодзи и хештегами

Подробное описание проекта и требований: [PROJECT.MD](./PROJECT.MD)

## Технологии

- **Next.js 14** - React фреймворк
- **OpenRouter API** - интеграция с AI (модель: deepseek/deepseek-chat)
- **Cheerio** - парсинг HTML статей
- **Tailwind CSS** - стилизация интерфейса

## Функциональность

- Парсинг статей с различных сайтов
- Три типа анализа через AI
- Обработка ошибок с понятными сообщениями
- Таймауты для предотвращения зависаний
- Оптимизированная производительность

## Установка

Установите зависимости:

```powershell
pnpm install
```

## Запуск

Запустите сервер разработки:

```powershell
pnpm dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Другие команды

- `pnpm build` - сборка приложения для продакшена
- `pnpm start` - запуск продакшен-версии
- `pnpm lint` - проверка кода линтером
